# 株式会社かみねんど コーポレートサイト

レトロゲーム風の1ページコーポレートサイト。RYU QUEST（osako.kaminendo.art）と同じ世界観。

- **公開先（予定）**: https://kaminendo.art/ （GitHub Pages + Cloudflare DNS）
- 構成: ヒーロー → 事業（魔法の公式LINE／パフォーマー派遣／商品開発／経営コンサル）→ 代表（RYU QUESTへリンク）→ 会社概要 → お問い合わせ
- お問い合わせは GAS Webアプリへ直接送信（osako@kaminendo.art へメール転送）、失敗時はメールアプリ起動
- 編集: `index.html` を直接編集して push すると本番反映
